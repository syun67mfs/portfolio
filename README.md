# ゲーム好きのゲーム好きによるゲーム好きの為のゲーム紹介サイト

## サイト概要

ゲームの作品と説明を紹介し、作品を検索するのは勿論それに対して評価やコメントが出来る。

### サイトテーマ

No Game No Life

### テーマを選んだ理由

自分がゲームが好きで新しいゲームを探す際にこの様なサイトがあれば色んなゲームやより良いゲームに出会えるのではないかと思った為。

### ターゲットユーザ

日本中のゲームを嗜む老若男女

### 主な利用シーン

どの様なゲームがあるかやどれが今人気なのかを調べたい時
